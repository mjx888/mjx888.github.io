Write to msh file

Wirte to msh file using function printMsh.
precision_nodecoor = 8;
file_name = 'test_1.msh';
printMsh( vert, tria, [], [], precision_nodecoor, file_name );

Mesh generated by function generateMesh

The mesh can also be generated by function generateMesh (in MATALB PDE toolbox).
Let's create a pde model based on polyshape object and use function generateMesh to generate FEM mesh. Then, we will write the mesh into msh file.

Create pde model based on polyshape object

First, we create polyshape (a square with a hole).
% Create unit square
vertex = [ 0 0; 1 0; 1 1; 0 1 ];
psUnitSq = polyshape(vertex);
% Scale with respect to centroid to create a tiny square
[x,y] = centroid(psUnitSq);
refpoint = [x,y];
scale_factor = 0.15;
psSqTiny = scale(psUnitSq, scale_factor, refpoint);
% boolean operation to create square with a hole
psSqHole = subtract(psUnitSq, psSqTiny);
plot(psSqHole);
axis equal
Convert to pde model
% triangulation representation of object
tr = triangulation( psSqHole );
% Create a pde model by using the triangulation data
model = createpde;
geometryFromMesh( model, tr.Points', tr.ConnectivityList' );
model
model =
PDEModel with properties: PDESystemSize: 1 IsTimeDependent: 0 Geometry: [1×1 DiscreteGeometry] EquationCoefficients: [] BoundaryConditions: [] InitialConditions: [] Mesh: [1×1 FEMesh] SolverOptions: [1×1 pde.PDESolverOptions]

Generate mesh

Use function generateMesh.
generateMesh( model );
model
model =
PDEModel with properties: PDESystemSize: 1 IsTimeDependent: 0 Geometry: [1×1 DiscreteGeometry] EquationCoefficients: [] BoundaryConditions: [] InitialConditions: [] Mesh: [1×1 FEMesh] SolverOptions: [1×1 pde.PDESolverOptions]
pdemesh( model );

Write to msh file

Extract mesh from pde model.
vert = model.Mesh.Nodes';
tria = model.Mesh.Elements';
Wirte to msh file using function printMsh.
precision_nodecoor = 8;
printMsh( vert, tria, [], [], precision_nodecoor );
printMsh Done! Check the msh file!
We will see a new file 'test.msh' in the current working folder of Matlab.
